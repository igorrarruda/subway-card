# Subway Card App

Subway Card App

## Get started

In the project directory run:

```cmd
yarn
```

to install dependences.

Then, run:

```cmd
yarn start
```

to run backend.

then, run

```cmd
yarn start:frontend
```

to run frontend.

## Testing

### Source-code tests

Run:

```cmd
yarn test
```

to run unit tests.

then, run

```cmd
yarn cucumber-test
```

to run cucumber tests.

finaly, run

```cmd
yarn selenium
```

to run cucumber tests (to execute this test correctly you need backend and frontend running in parallel).
